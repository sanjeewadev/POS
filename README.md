# Touch-Friendly POS System (React + Electron)

A high-performance, cross-platform desktop Point of Sale (POS) application designed specifically for touch interfaces. 

Unlike traditional, clunky web-based POS systems, this application leverages **Electron** to provide native desktop performance and hardware access, while utilizing **React** and **Material UI** to deliver a responsive, touch-optimized user experience for fast-paced retail environments.

### Tech Stack & Architecture
* **Programming Language:** TypeScript
* **Frontend:** React.js
* **Desktop Wrapper:** Electron.js (for native OS integration and performance).
* **Database:** SQLite

### Key Features
* **Touch-Optimized UI:** Designed with large, high-contrast buttons and no tiny icons, ensuring fast checkout times on touch-screen monitors.
* **Custom Calculator Logic:** Built-in dynamic calculator components for rapid price adjustments and tender inputs.
* **State Management:** Clean sidebar logic and complex state handling for real-time cart updates and "Today's Sales" reporting.
* **Native Desktop Performance:** Packaged via Electron to run smoothly as a standalone app without relying on a web browser.
